# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-07

### Added
- Initial release of QuantumPytho
- Physically correct Bloch sphere state projection
- QRNG module with golden-ratio scaling
- Circuit explorer (Bell pairs, Hadamard sweeps)
- Teleportation protocol skeleton with scientific references
- VQE framework (generic optimizer + physical H2 VQE)
- CLI menu with modular architecture
